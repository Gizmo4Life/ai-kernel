# Bootstrap

**Bootstrap** refers to the self-starting process that builds up a complex system.

## Kernel Bootstrap

The AI Kernel was "bootstrapped" by first defining its own core standards (standard-file-standard, glossary-entry-standard) and then using those standards to create the initial set of glossary terms and skills. This self-referential process ensured that the framework was functional and validated from its very first commit.
