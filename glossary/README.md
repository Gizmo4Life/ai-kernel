# Glossary

This directory contains the single source of truth for all concepts and terminology used within the AI Kernel.

## Core Concepts

- [Glossary Entry](glossary-entry.glossary.md): The fundamental unit of knowledge.
- [Standard](standard.glossary.md): Guidelines governed by the PADU scale.
- [Skill](skill.glossary.md): Atomic, tool-centric actions.
- [Instruction](instruction.glossary.md): Multi-step workflows.
- [Agent](agent.glossary.md): Autonomous actor definitions.

## Metadata & Framework

- [Frontmatter](frontmatter.glossary.md): YAML metadata for machine discovery.
- [PADU Scale](padu-scale.glossary.md): The rating system for standards.
- [Progressive Disclosure](progressive-disclosure.glossary.md): The optimization strategy for agent scanning.
- [Context](context.glossary.md): Persistent conversation and project state.
- [Delegation](delegation.glossary.md): Mechanism for agent collaboration.
- [Antipattern](antipattern.glossary.md): Identifying and avoiding ineffective practices.

## System Architecture

- [Knowledge Graph](knowledge-graph.glossary.md): The interconnected network of kernel content.
- [Knowledge Map](knowledge-map.glossary.md): Curated views of the Knowledge Graph for specific tasks.
- [Heuristics](heuristics.glossary.md): Logic and shortcuts for agent decision-making.
- [Bootstrap](bootstrap.glossary.md): The self-starting initialization process.
- [Authority](authority.glossary.md): Permission levels for agents (Propose vs Suggest).
- [Quality Gate](quality-gate.glossary.md): Validation checkpoints in workflows.
- [Atomicity](atomicity.glossary.md): The single-responsibility principle for skills.
- [Orchestration](orchestration.glossary.md): The coordination of skills into instructions.
