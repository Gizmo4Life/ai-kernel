# Knowledge Map

A **Knowledge Map** is a subset of the **Knowledge Graph**. While the graph contains everything, a map highlights only the standards, skills, and glossary entries relevant to a specific problem.


## Utility

- **Onboarding**: Helps new users understand which parts of the kernel apply to their current task.
- **Guidance**: The **Librarian** agent generates these maps to answer "How do I do X?" queries.
